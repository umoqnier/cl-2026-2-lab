# Práctica 2: Estadisticas de lenguaje y diversidad

## Requisitos

- Python >= 3.12

- Jupyter Notebook o JupyterLab (o algún otro entorno para ejecutar notebooks).

## Dependencias

Para ejecutar el notebook `Practica2_Estadisticas_De_Lenguaje_y_Diversidad.ipynb` se requieren:

- pandas
- numpy
- matplotlib
- scipy
- plotly
- elotl
   
## Ejecución

Abre el archivo `Practica2_Estadisticas_De_Lenguaje_y_Diversidad.ipynb` en algun entorno capaz de ejecutar notebooks y ejecuta las celdas en orden.

> **Nota:**
> - El notebook usa archivos locales en `data/`: `languages_and_dialects_geo.csv` y `languoid.csv`.
> - Para cargar el corpus de `elotl` puede requerirse conexión a internet la primera vez.
